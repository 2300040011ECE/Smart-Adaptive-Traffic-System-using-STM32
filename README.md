A Smart Adaptive Traffic System using STM32 adjusts traffic light timing based on vehicle density (using IR/ultrasonic sensors) instead of fixed delays.

Below is a complete example project using:

🔹 STM32 (e.g., STM32F103C8T6 – Blue Pill)

🔹 4 IR vehicle sensors (one per lane)

🔹 Traffic LEDs (Red, Yellow, Green per lane)

🔹 STM32 HAL library (CubeIDE)
